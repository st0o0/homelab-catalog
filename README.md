# homelab-catalog

Private Dockhand template catalog for homelab services.

## Usage

In Dockhand UI: **Templates → Sources → Add**

```
Name: Homelab
URL:  https://raw.githubusercontent.com/st0o0/homelab-catalog/main/templates.json
```
